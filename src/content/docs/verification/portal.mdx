The portal is a Vite SPA hosted on <Tooltip term="amplify" label="Amplify" />. Verify it loads in a browser and can query <Tooltip term="tec" label="TEC" /> data after [CORS Lockdown](/aws-event-driven-serverless-walkthrough/deploy/cors-lockdown/).

## Open the portal

From the demo repo root:

```bash
terraform -chdir=terraform output -raw app_url
```

Open the URL in a browser.

## Expected behavior

1. **Station browser** — Alphabetically sorted <Tooltip term="station" label="stations" /> with search (Enter-to-select)
2. **Auto-load** — Selecting a station loads the most recent available date
3. **Time-series chart** — Density-adaptive rendering; legend hidden above 12 satellites with a hint to use the Satellite filter
4. **IPP map** — Outlier-resistant fitBounds viewport, dateline-safe longitude normalization
5. **Reprocess panel** — Submit jobs with parameter overrides

## Prerequisites

- Processed output exists under `processed/tec/` (see [S3 Processed Output](/aws-event-driven-serverless-walkthrough/verification/s3-processed-output/))
- [CORS Lockdown](/aws-event-driven-serverless-walkthrough/deploy/cors-lockdown/) applied so browser requests succeed
- Portal deployed via Terraform or [Amplify Portal](/aws-event-driven-serverless-walkthrough/deploy/amplify-portal/) manual script
