  <thead>

```json
{
  "data": [
    {
      "epoch": "2024-05-29T01:00:00Z",
      "sv": "G01",
      "id_arc": 1,
      "lat_ipp": -36.85,
      "lon_ipp": 174.76,
      "azi": 45.2,
      "ele": 30.1,
      "bias": 0.5,
      "stec": 12.3,
      "vtec": 8.7,
      "veq": 9.1
    }
  ],
  "meta": {
    "row_count": 1,
    "truncated": false
  }
}
```

## Catalog response — stations (HTTP 200)

`GET /catalog`

```json
{
  "stations": ["AUCK", "AVLN", "WGTN"]
}
```

## Catalog response — dates (HTTP 200)

`GET /catalog?station=AUCK`

```json
{
  "dates": [
    { "year": 2024, "doy": 150 },
    { "year": 2024, "doy": 149 }
  ]
}
```

## Error responses

### HTTP 400

```json
{
  "error": "Missing required parameter: station"
}
```

### HTTP 404

```json
{
  "error": "Job not found: \{job_id\}"
}
```

### HTTP 500

```json
{
  "error": "Internal server error"
}
```

Catalog endpoints return HTTP 500 when `DATA_LAKE_BUCKET` is missing or when <Tooltip term="s3" label="S3" /> listing fails.

```json
  "created_at": "2024-05-29T00:00:00Z",
  "updated_at": "2024-05-29T00:05:00Z"
}
```

## Example curl commands

```bash
export API_URL="$(terraform -chdir=terraform output -raw api_url)"

curl "${API_URL}/catalog"
curl -s "${API_URL}/catalog?station=auck" | jq .
curl -s "${API_URL}/query?station=auck&start_time=2026-06-28T00:00:00Z&end_time=2026-06-28T23:59:59Z&sv=G01" | jq .meta
```

See [REST API Usage](/aws-event-driven-serverless-walkthrough/usage/rest-api-usage/) for operational guidance.
