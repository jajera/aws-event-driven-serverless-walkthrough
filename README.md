# aws-event-driven-serverless-walkthrough

Step-by-step documentation site for deploying the [event-driven-serverless-platform-demo](https://github.com/jajera/event-driven-serverless-platform-demo) GNSS platform on AWS.

Published at [jajera.github.io/aws-event-driven-serverless-walkthrough](https://jajera.github.io/aws-event-driven-serverless-walkthrough/).

## Local development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run validate   # prettier + markdownlint
npm run build      # astro build (also npm test)
npm run generate:diagram   # fetch AWS icons and rebuild architecture-diagram.svg
```
